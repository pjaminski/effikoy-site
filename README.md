## 💡 About

Effikoy is my personal brand – a combination of *efficiency* and *real effect*.  
In Esperanto, **“efiko”** means *effect* (plural: *efikoj* – outcomes).  
That’s exactly what defines my engineering mindset:  
**Efficient Code. Real Effect.**

This site is a minimalist, single-page static site built with semantic HTML, modular CSS, and vanilla JavaScript - no frameworks, no build steps - deployed directly on GitHub Pages with a custom domain.

---

## 🧱 Tech Stack

- **HTML5 + CSS3 + JavaScript (ES6)**  
- Simple section-based navigation with hash routing (`#home`, `#about`, `#contact`)  
- Responsive header with hamburger menu for mobile  
- No external dependencies or frameworks  
- Hosted on **GitHub Pages**  
- Custom domain via **OVH DNS + GitHub CNAME**
